# bkt2gh Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Go CLI that migrates selected Bitbucket Cloud repositories to GitHub.

**Architecture:** Keep CLI parsing thin and delegate to focused internal packages. Use REST clients with injectable HTTP clients and git operations with injectable command runners for tests.

**Tech Stack:** Go 1.22, standard library, git CLI, optional git-lfs CLI.
